# Suno: flera fight-låtar (prompter)

Engelska prompter för stridsmusik till western/pixel-roguelike. Använd i Suno; lägg till i **Style** om du har det, t.ex. `western, combat, instrumental`.

Exporterade filer kan ligga under t.ex. `assets/music/combat/`.

---

## 1. Standard strid (exploration combat, loop-vänlig)

```
High-energy spaghetti western combat music for a retro pixel roguelike, galloping rhythm, tremolo electric guitar, driving acoustic strums, harmonica stabs, brushed snare and kick, upright bass, mid-fast tempo, tense but heroic, instrumental only, no vocals, no EDM
```

## 2. Tätare / mer “arcade shooter”

```
Fast western action arcade soundtrack for a 2D cowboy shooter, relentless percussion, sharp electric guitar riffs, occasional brass stabs, minor-key tension, punchy mix suitable for looping gameplay, instrumental only, no vocals
```

## 3. Mini-boss / elit-möte (tyngre, mer dramatik)

```
Epic western boss encounter music for a pixel game, slower heavy groove opening into faster passages, distorted guitar layers, dramatic low brass, tribal drums, outlaw desert atmosphere, cinematic but game-mix friendly, instrumental only, no vocals
```

## 4. Doom-metal × western

```
Industrial doom western metal combat theme, downtuned guitar chugs, double-kick bursts, sub bass, distant harmonica through distortion, galloping transitions, dark frontier mood, instrumental only, no vocals, not EDM
```

## 5. “Bullet ballet” / Dead Eye-känsla (lite mer tomrum, fokus)

```
Sparse tense western duel combat bed for a video game, minimal drums with sudden accents, single-note guitar tension, suspenseful harmonica, heartbeat-like low pulse, mid tempo with bursts, instrumental only, no vocals
```

## 6. Kortare “sting” / wave clear

```
Victory sting idea: bright major-key spaghetti western flourish, ragtime-adjacent piano fragment, fiddle lift, 8–12 seconds feel, retro game fanfare energy, instrumental only, no vocals
```

*(Om Suno alltid gör 1–2 min: använd som intro till en längre track eller generera och klipp.)*

---

## Tips när du genererar

- **Instrumental:** behåll `instrumental only, no vocals` i varje prompt.
- **Om det blir för “film”:** lägg till `retro game mix`, `mid-forward drums`, `not orchestral`.
- **Om det blir för EDM:** lägg till `no electronic dance beats, no four-on-the-floor club kick`.
